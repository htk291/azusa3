# NAME

AZUSA 3 - AZUSA theme for reveal.js

# HOW TO USE

`<h1>` aligned center.
`<h2>` aligned to the left.

You can set bg-color and color like this.
`<!-- .slide: data-state="navy" class="navy" -->`

# COLORS

* GREEN (default)
    - green
    - navy
    - beige

Other colors are comming soon ...
